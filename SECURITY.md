# Security Policy

## Supported Versions

Only the latest stable release builds are supported with security patches.

## Reporting a Vulnerability

If you identify a security vulnerability in the codex-plugin-cc integration (such as prompt manipulations causing local shell breakouts, token/credentials hijack during review operations, or workspace files exposure), please do **not** open a public issue.

Please report vulnerabilities privately by emailing the maintainers or using GitHub's private vulnerability reporting feature.
